We can the style= attribute on HTML tags to apply various styles to text and blocks of text.

While the style attribute, works it leads to overly verbose HTML so it is more typical to write CSS style rules to apply styles to many tags all at once.

We have a special span tag just so we can apply CSS to a bit of text.

The "Cascading" in CSS means that the closest style setting (monospace) "wins" over a syle setting (arial) that is "further away".

We can put a border around a block of text. You probably want a CSS cheat sheet handy or use Google to search to find CSS values.
